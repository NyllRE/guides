1. Task List

1.1. TODO Make an Emacs Org GUIDE

  • SPC-m-t-# marks a line as TODO
  • SPC-m-d-s opens org-schedule for a section
  • RET => toggle TODO & DONE

1.2. TODO do multiple things in one task [1/5] [20%]

  • [ ] CTRL + RET => add new item
  • [ ] C-c C-c => toggle Box
  • [X] SPC-m-x => same thing (space mark X)
  • [ ] add [0/0] to the head => automatically adds [0/X]
  • [-] for sub tasks
    • [-] you can do them easily
    • [ ] when they’re finished the one up is finished

1.3. PROJ Org Agenda SPC-o-A

  • a = agenda for current week (what you scheduled)
  • t = todo items not done
    • t => toggle state
  • s = search by keyword
    • you can use regex!

1.4. PROJ SPC-X for Capture Mode

  • j => Journal
  • n => notes
  • t => to-do’s
  • p => local
    • to-do’s
    • notes
    • change-log

1.5. PROJ C-c C-e for exporting

  • m m => markdown
  • h h => html
